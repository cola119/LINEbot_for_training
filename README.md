# トレーニング管理用LINE bot
トレキャンサイトを開いて入力するのが難しい人向けLINE bot.  
LINEグループトークで決まった文法で発言すると、データベースに保存してくれる。

# Dependency
PHP / LINE Messaging API


# Usage
トレキャン[ID]  
ランニング[距離]  
option:[ごじらん]

#### 例(ユーザー1,ランニング10km)
トレキャン1  
ランニング10
